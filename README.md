# AutoDaily —— 原神每日自动监控

> 🎮 电脑空闲时自动执行 BetterGI 一条龙，完美解放双手。
>
---

## 这是什么？

AutoDaily 是一个 Windows 桌面工具。它在你离开电脑后，自动帮你完成原神的每日任务（通过 BetterGI）。

**你只需要**：下载 → 双击 → 跟着向导点几下 → 完成。之后完全自动化。

---

## 功能

| 功能 | 说明 |
|---|---|
| 🕐 空闲检测 | 检测到电脑超过设定时间无键鼠操作（默认15分钟） |
| ⏰ 间隔控制 | 两次执行至少间隔设定时间（默认20小时，保证树脂恢复） |
| 🔇 自动静音 | 执行期间可选停止 Windows Audio 服务，不会被游戏声音打扰 |
| 🔍 自动检测 | 首次运行自动搜索 BetterGI 安装位置和一条龙配置 |
| 📊 状态仪表盘 | 主窗口实时显示监控状态、间隔、空闲时长 |
| 📝 执行日志 | 每次触发、状态变更都有中文日志记录 |
| 🖥️ 系统托盘 | 关闭窗口后驻留托盘，右键菜单操作 |
| 🚀 开机自启 | 可设置开机自动启动（静默模式） |
| 🎨 定制图标 | 萌萌的宵宫图标，托盘、窗口、文件资源管理器都能看到 |

## 快速开始

### 1. 下载

从 [Releases](https://github.com/你的用户名/AutoDaily/releases) 页面下载 `AutoDaily.exe`（单文件，约 155 MB，无需安装 .NET）。

### 2. 运行

双击 `AutoDaily.exe`。首次运行会自动弹出**设置向导**：

1. **欢迎页** → 看到宵宫头像，点击下一步
2. **自动检测 BetterGI** → 程序会自动搜索你电脑上的 BetterGI
3. **选择一条龙配置** → 选择 BetterGI 中要使用的任务配置（默认即可）
4. **设置参数** → 调整空闲阈值、执行间隔、是否静音、是否开机自启
5. **完成** → 监控立即启动

### 3. 使用

- **主窗口**：查看监控状态、手动触发、修改设置
- **关闭窗口**：不会退出，最小化到系统托盘
- **托盘图标**：右键菜单 → 启动/停止/触发/设置/退出
- **双击托盘**：显示主窗口

## 工作原理

```
电脑空闲 ≥ 15分钟 + 距上次执行 ≥ 20小时
        │
        ▼
  静音系统（可选）→ 启动 BetterGI 一条龙 → 等待完成 → 恢复声音
```

- BetterGI 必须已配置好一条龙任务
- 原神需要在 BetterGI 启动前已运行（BetterGI 会自动启动原神，取决于你的配置）

---

## 配置说明

| 参数 | 默认值 | 说明 |
|---|---|---|
| 空闲触发阈值 | 15 分钟 | 键鼠无操作超过此时间才可能触发 |
| 最小执行间隔 | 20 小时 | 两次执行的最小间隔（原神树脂恢复时间） |
| 执行期间静音 | 开启 | 停止 Windows Audio 服务（需管理员权限） |
| 开机自动启动 | 关闭 | 开机后自动启动监控（静默模式） |

## 常见问题

**Q: 为什么电脑空闲了但没触发？**
A: 两个条件必须同时满足：空闲 ≥ 15 分钟 **且** 距上次执行 ≥ 20 小时。打开主窗口查看当前间隔和空闲时长。

**Q: 静音需要管理员权限，会弹 UAC 吗？**
A: 每次执行时如果开启静音，会弹出一次 UAC 窗口请求权限。如果你希望在离开电脑时自动通过，可以将 UAC 级别降低或关闭静音功能。

**Q: BetterGI 找不到怎么办？**
A: 在设置中点击「📂 浏览」手动选择 BetterGI 安装目录。BetterGI 是便携软件，没有固定安装位置。

**Q: 如何卸载？**
A: 删除 `AutoDaily.exe` 即可。如果开启了开机自启，先在设置中关闭，或手动删除注册表 `HKCU\Software\Microsoft\Windows\CurrentVersion\Run\AutoDaily`。
